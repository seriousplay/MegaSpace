## 设计原则
1. **以用户为中心**: 围绕教师、学生、家长三个核心用户群体设计
2. **AI优先**: 将AI能力深度集成到每个功能模块
3. **安全可靠**: 参考Magic School的安全标准，确保数据保护
4. **可扩展性**: 支持未来功能扩展和用户规模增长
5. **本土化**: 结合中国教育特色和用户习惯
